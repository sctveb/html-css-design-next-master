■ 다운로드 데이터 내용에 대해서

● 이 책의 샘플은 index.html 부터 확인하면 됩니다. 브라우저로 index.html을 열어 확인해주세요.

● 샘플 데이터는 제작 단계별로 저장되어 있습니다.
예) STEP2-1-2의 데이터는 "2-1-2" 폴더에 들어 있습니다.

● 각 페이지의 최종적인 완성 데이터는 "sample" 폴더에 들어있습니다.

● "Web" 폴더는 STEP1-3에서 사용하는 폴더입니다.
샘플 제작 시작 때 활용하기 바랍니다.

● "img" 폴더에는 샘플에서 사용하는 이미지가 들어있습니다.
※이미지는 Pixabay(https://pixabay.com/)에서 가져왔습니다.

● HTML/CSS 폴더는 UTF-8 인코딩으로 저장되어 있습니다. UTF-8을 지원하는 텍스트 에디터를 사용해서 열어주세요.
